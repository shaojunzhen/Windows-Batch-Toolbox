# Windows-Batch-Toolbox

A simple Windows batch toolbox made by a junior high school student (beginner), with many issues, suggestions are welcome.

\# 简易Windows批处理工具箱 (Windows Batch Toolbox)

v2.5



\## 介绍 (Introduction)

一个纯手动编写的Windows批处理脚本工具箱，包含计算机工具启动、系统信息查询、简易系统操作等基础功能。

A manually written Windows batch script toolbox, including basic functions such as computer tool startup, system information query, and simple system operations.



\### 关于作者 (About the Author)

本人是一名初中生，刚接触批处理编程不久，写这个小工具仅为学习和练手目的。代码中存在大量bug、逻辑不严谨、功能不完善的问题（比如偶发闪退、部分功能需要管理员权限才能运行、交互体验差等），代码规范也完全不达标。

I am a junior high school student who has just started learning batch programming. This small tool is only made for learning and practice. There are many problems in the code (such as occasional crashes, some functions requiring administrator privileges, poor interaction experience, etc.), and the code specification is not up to standard at all.



\### 核心功能 (Core Functions)

1\. 快速启动系统自带工具（放大镜、画图、记事本、CMD等）

2\. 查询系统信息（IP、端口、WiFi信息、磁盘/内存占用等）

3\. 简易系统操作（关机/重启、修改密码、清理缓存、结束进程等）

1\. Quickly launch system-built tools (Magnifier, Paint, Notepad, CMD, etc.)

2\. Query system information (IP, ports, WiFi info, disk/memory usage, etc.)

3\. Simple system operations (shutdown/restart, password modification, cache cleaning, process termination, etc.)



\### 使用方法 (Usage)

1\. 下载 `Toolbox\_v2.5\_en.bat`（英文版本）或中文版本bat文件

2\. 右键以管理员身份运行（部分功能需要管理员权限）

3\. 按照界面提示输入数字即可使用对应功能

1\. Download `Toolbox\_v2.5\_en.bat` (English version) or the Chinese version bat file

2\. Right-click and run as administrator (some functions require admin privileges)

3\. Enter numbers according to the interface prompts to use corresponding functions



\### 已知问题 (Known Issues)

\- 部分功能仅支持Windows 7及以上系统

\- 无输入校验，输错数字会导致逻辑异常

\- 本地组策略编辑器配置功能可能在部分系统失效

\- Some functions only support Windows 7 and above

\- No input validation, wrong numbers will cause logical exceptions

\- The Local Group Policy Editor configuration function may fail on some systems
